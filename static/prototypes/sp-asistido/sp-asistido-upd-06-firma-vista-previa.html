UPDCUS00

Nombre largo (descriptivo) (máx. 50)

Actualización de cliente por clave (UPDATE)

Comentario descriptivo (opcional)

Actualiza datos de cliente con WHERE obligatorio y opción de concurrencia.

SPECIFIC (objeto interno de catálogo IBM i)

UPDCUS0SP1

Alineado con la cláusula SPECIFIC del CREATE OR REPLACE PROCEDURE.

Concurrencia (opcional, según paso 5)

Si se activa, el SQL agrega condición extra de versión/timestamp en el WHERE para evitar sobrescribir cambios concurrentes.

Parámetros IN (SET + WHERE)

P_CUSNBR DECIMAL(9), P_CUSNAME VARCHAR(40)

Parámetros OUT (plantilla CODAS §9.2)

P_ERROR_CODE INTEGER, P_ERROR_MSG VARCHAR(200)

CREATE OR REPLACE PROCEDURE R1DATA.UPDCUS00 (
    IN  P_CUSNBR DECIMAL(9),
    IN  P_CUSNAME VARCHAR(40),
    OUT P_ERROR_CODE INTEGER,
    OUT P_ERROR_MSG  VARCHAR(200)
)
LANGUAGE SQL
MODIFIES SQL DATA
SPECIFIC UPDCUS0SP1
BEGIN
    DECLARE v_sqlstate CHAR(5) DEFAULT '00000';
    DECLARE v_msg      VARCHAR(200) DEFAULT '';

    DECLARE EXIT HANDLER FOR SQLSTATE '23505'
    BEGIN
        SET P_ERROR_CODE = 100;
        SET P_ERROR_MSG  = 'Restricción de unicidad.';
    END;

    DECLARE EXIT HANDLER FOR SQLEXCEPTION
    BEGIN
        GET DIAGNOSTICS CONDITION 1
            v_sqlstate = RETURNED_SQLSTATE,
            v_msg = MESSAGE_TEXT;
        SET P_ERROR_CODE = -1;
        SET P_ERROR_MSG  = 'Error SQLSTATE=' CONCAT v_sqlstate CONCAT ' - ' CONCAT v_msg;
    END;

    UPDATE R1DATA.CUSMAS
       SET CUSNAME = P_CUSNAME
     WHERE CUSNBR = P_CUSNBR;

    SET P_ERROR_CODE = 0;
    SET P_ERROR_MSG  = 'OK';
END;
